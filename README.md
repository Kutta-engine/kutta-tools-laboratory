# Tools

## Image Batch Resizer 600px

画像を一括で長辺600pxにリサイズできるシンプルなGUIツールです。

### Features

- 長辺600pxへ一括変換
- 縦横比維持
- 小さい画像はそのままコピー
- 元画像を変更しない
- フォルダ単位で一括処理

### Environment

- macOS
